# Field Command Centre

A fully-featured construction site management tool built for Resident Engineers, Site Managers, and Project Managers. Runs entirely in the browser — no installation, no subscription, no internet required.

## Live Demo
🔗 [fieldcommandke.netlify.app](https://fieldcommandke.netlify.app)

## Features

- **Programme Tracker** — Phase timeline with % complete and status flags
- **Work Package Register** — Trade, assignee, phase linkage, inline editing
- **Risk Register** — RAG-coded risks with probability, mitigation, and phase tagging
- **QA/QC Snagging** — Raise and track defects through full remediation lifecycle
- **Site Diary** — Daily log with weather, workforce, activities, and inspections
- **Team Utilisation** — Per-person load tracking with visual colour coding
- **Materials Tracker** — Delivery and stock level monitoring
- **RE Report Generator** — Auto-compiled Resident Engineer report, print-ready
- **Excel Export** — One-click .xlsx export of any register

## How to Use

1. Download `index.html`
2. Open it in any modern browser (Chrome, Firefox, Safari, Edge)
3. No server needed — runs 100% locally

## Tech Stack

- React 18 (via CDN, no build step)
- SheetJS for Excel export
- Browser localStorage for offline data persistence
- Pure HTML/CSS/JS — single file deployment
